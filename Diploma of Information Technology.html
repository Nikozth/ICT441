Diploma of Information Technology:

  • Computer Systems and Hardware: Students will learn the basics of computer system architecture, hardware components, and operating systems.
  • Network Technology: It includes the basics of network, network topology, protocols and network security.
  • Database Management: Students may learn the basics of database design, query language, and administration.
  • Web Development: Students may learn how to design and develop websites, including front-end and back-end technologies.
  • Information Security: Provides basic knowledge of information security principles, security measures, and security practices.
